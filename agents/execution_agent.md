# ExecutionAgent 명세

## 목적
입찰 당일 이전에 필요한 준비 상태를 점검하고 누락 항목을 차단한다.

## 입력
- 게이트 통과 후보
- 입찰 일정 정보
- 사용자 준비 상태 체크리스트

## 출력
- 실행 체크리스트
- 일정 알림 항목
- `execution_status`: READY | HOLD

## 판단 규칙
- 현장조사/서류/자금조달 3개 항목 중 하나라도 미완료면 `HOLD`.
- 일정 임박(예: 48시간 이내) 미완료 항목은 우선 알림한다.
- 최종 승인 기록이 없으면 입찰 진행 신호를 생성하지 않는다.

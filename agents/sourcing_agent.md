# SourcingAgent 명세

## 목적
법원 경매 공고에서 예산과 지역 조건에 맞는 1차 후보 물건을 자동 생성한다.

## 입력
- 시장 데이터: 사건번호, 물건종류, 감정가, 최저가, 매각기일, 소재지
- 사용자 조건: 예산 상한, 선호 지역, 회피 유형

## 출력
- `candidate_list`:
  - `candidate_id`
  - `base_score`
  - `price_band_fit`
  - `liquidity_hint`

## 판단 규칙
- 총비용 추정치가 예산 범위 밖이면 후보에서 제외한다.
- 표준화된 주거형 물건에 우선 점수를 부여한다.
- 데이터 누락(핵심 필드) 발생 시 `NEEDS_REVIEW` 태그를 붙인다.

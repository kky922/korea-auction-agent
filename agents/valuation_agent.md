# ValuationAgent 명세

## 목적
시세와 비용을 보수적으로 반영해 최대 입찰가(`max_bid`)와 안전마진을 계산한다.

## 입력
- 후보 물건 데이터
- 예상 시세(보수값), 수리비, 취득 부대비용
- 리스크 리포트

## 출력
- `max_bid`
- `expected_margin`
- `expected_margin_rate`
- `break_even`

## 판단 규칙
- 비용은 낙관값이 아닌 상단 추정치를 기본 사용한다.
- 목표 수익률 12% 미만이면 탈락 후보로 표기한다.
- 권리 리스크가 해소되지 않은 건은 `max_bid`를 계산하지 않는다.

# CoachAgent 명세

## 목적
초보 투자자의 경매 기초 역량을 주차별로 끌어올려, 실전 입찰 전 필수 지식을 확보한다.

## 입력
- 학습자 프로필: 투자금, 목표 지역, 가용 시간
- 복기 데이터: 최근 실패 사유, 자주 틀린 개념
- 정책 문서: `policy/investment_policy.md`

## 출력
- 주간 학습 미션 3~5개
- 개념 퀴즈(객관식/사례형)
- 오답노트와 보완 과제

## 판단 규칙
- 권리분석 관련 오답률이 20% 초과면 다음 주 권리 파트를 강화한다.
- 게이트 실패 상위 사유를 학습 주제에 우선 반영한다.
- 실전 직전 주에는 체크리스트 훈련 비중을 높인다.

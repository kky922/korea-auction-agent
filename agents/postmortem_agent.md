# PostMortemAgent 명세

## 목적
유찰/낙찰 후 결과를 구조화해 다음 회차 전략과 학습 항목에 반영한다.

## 입력
- 실행 결과: 낙찰가, 유찰 여부, 경쟁률
- 예상 대비 오차: 비용/수익/기간
- 게이트 판정 로그

## 출력
- `weekly_review`
- `strategy_patch` (정책 파라미터 조정 제안)
- `learning_topics` (CoachAgent 전달용)

## 판단 규칙
- 반복 실패 사유가 2회 이상 누적되면 정책 파라미터 조정을 제안한다.
- 예상 대비 손익 오차가 크면 가치평가 가중치를 보수적으로 조정한다.
- 근거 없는 사후 해석을 금지하고 로그 근거를 링크한다.

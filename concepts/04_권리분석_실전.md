# 권리분석 실전

## 핵심 개념

등기부등본 + 법원 경매 서류를 조합해서
"이 물건 낙찰받으면 실제로 얼마 드는지, 위험은 없는지" 판단하는 것.

---

## 확인해야 할 서류

| 서류 | 어디서 | 확인할 것 |
|------|--------|----------|
| 등기부등본 (을구) | 대법원 인터넷등기소 | 근저당, 전세권, 가압류 날짜와 금액 |
| 등기부등본 (갑구) | 동일 | 압류, 가처분, 소유권 변동 |
| 임차인 현황조사서 | 법원 경매 사이트 | 임차인 전입일, 확정일자, 보증금 |
| 감정평가서 | 법원 경매 사이트 | 감정가, 물건 상태 |
| 매각물건명세서 | 법원 경매 사이트 | 인수해야 할 권리 요약 (법원이 정리해줌) |

---

## 등기부등본 읽는 순서

1. **을구에서 말소기준권리 찾기**
   - 근저당, 가압류, 압류 중 가장 오래된 것
   - 그 날짜를 기준점으로 설정

2. **갑구에서 압류/가처분 확인**
   - 말소기준권리 이전 것 있으면 인수 대상

3. **말소기준권리 이전 권리 있는지 확인**
   - 있으면 → 인수 부담 계산
   - 없으면 → 안전

---

## 매각물건명세서 활용법

법원이 권리분석을 어느 정도 해서 정리해둔 문서.
**"인수할 권리"** 항목을 먼저 봄.

- 비고란에 아무것도 없으면 → 비교적 안전
- "임차인 ○○○은 대항력 있음" → 주의
- "유치권 신고 있음" → 주의

> 단, 법원도 실수하거나 누락할 수 있으니 맹신 금지.

---

## 실전 물건 분석 예시

### 등기부등본
```
[갑구]
2019.05.10  소유권이전 (홍길동)

[을구]
2020.03.15  근저당 ○○은행 120,000,000원   ← 말소기준권리
2022.07.20  가압류 채권자 A 20,000,000원
```

### 임차인 현황조사서
```
임차인 김철수
전입일: 2021.06.01
확정일자: 2021.06.05
보증금: 50,000,000원
점유: 실거주 중
```

### 분석

| 항목 | 내용 |
|------|------|
| 말소기준권리 | 2020.03.15 근저당 |
| 김철수 전입 | 2021.06.01 → 말소기준권리 이후 |
| 김철수 대항력 | 없음 |
| 김철수 우선변제권 | 있음 (후순위) |
| 인수할 권리 | 없음 |

**낙찰가 1억 5천 기준 배당 시뮬레이션:**
```
경매비용          약 300만
근저당 ○○은행    1억 2천만  → 전액 수령
가압류 A          남은 돈에서 배당
김철수            배당 남으면 수령, 없으면 명도 대상
```

→ 김철수가 배당 못 받으면 명도 필요하지만 대항력 없으니 협의 후 이사비 주고 내보내면 됨. **비교적 안전한 물건.**

---

## 위험 신호 체크리스트

- [ ] 말소기준권리 이전 전입 임차인 있음 → 보증금 인수 가능성
- [ ] 확정일자 없는 선순위 임차인 → 대항력만 있어서 전액 인수
- [ ] 유치권 신고 → 현장 확인 필수
- [ ] 가처분 등기 → 소유권 분쟁 가능성
- [ ] 법정지상권 가능성 → 토지/건물 소유자 다른 경우
- [ ] 감정가 대비 낙찰가가 너무 낮은 이유 있는지 확인

---

## 실제 입찰 전 계산식

```
낙찰 예상가
- 경매 비용 (낙찰가의 약 1~2%)
- 취득세 (주택: 1~3%, 비주택: 4%)
- 인수해야 할 권리 금액
- 명도 비용 (이사비 등)
- 수리비 (현장 답사 후 추정)
= 실제 취득 원가
```

이게 시세보다 충분히 낮아야 수익 있음.

# 수익계산

## 핵심 개념

낙찰 전에 시나리오별 수익을 미리 계산해서
"이 가격에 낙찰받으면 얼마 남는가" 확인하는 것.

---

## 두 가지 시나리오

### 1. 단기 매도 (플리핑)
낙찰 → 수리 → 매도

### 2. 임대 (월세/전세)
낙찰 → 수리 → 임차인 구함 → 보유

---

## 단기 매도 수익 계산

```
매도가
- 낙찰가
- 취득세
- 수리비
- 명도 비용
- 대출 이자 (보유 기간)
- 매도 중개수수료
- 양도소득세
= 순수익
```

**예시:**
```
매도가          3억
낙찰가         -2억 5천
취득세         -250만 (1%)
수리비         -500만
명도 비용       -100만
대출이자        -300만 (3개월 보유)
중개수수료      -270만 (매도가의 0.9%)
양도세         -약 500만 (1년 미만 단기, 세율 높음)
= 순수익       약 2,080만
```

---

## 임대 수익 계산

### 월세 수익률
```
연 임대수익 / 실투자금 × 100 = 수익률
```

**예시:**
```
낙찰가         1억 8천
대출           1억 2천 (LTV 70% 가정)
실투자금        6천만 (자기 돈)

월세           65만 / 월
연 임대수익     780만
대출이자        연 360만 (3% 가정)
순 임대수익     420만

수익률 = 420만 / 6천만 × 100 = 7%
```

---

## 전세 수익 계산 (레버리지)

```
낙찰가         1억 8천
전세가         1억 5천
실투자금        3천만 (낙찰가 - 전세금)

→ 3천만으로 부동산 1억 8천짜리 보유
→ 추후 매도 시 시세차익 노림
```

갭투자 방식. 현금흐름은 없지만 소액으로 부동산 보유 가능.

---

## 주의할 비용들

| 항목 | 금액 | 비고 |
|------|------|------|
| 취득세 | 낙찰가의 1~3.5% | 주택 수에 따라 다름 |
| 등록세 | 취득세에 포함 | |
| 법무사 비용 | 50~100만 | 소유권 이전 등기 |
| 대출 이자 | 연 3~5% | 보유 기간 내내 발생 |
| 수리비 | 100만~수천만 | 현장 답사 후 추정 |
| 양도세 | 단기 높음, 2년 이상 일반세율 | 보유 기간 중요 |

---

## 양도세 핵심

| 보유 기간 | 세율 |
|----------|------|
| 1년 미만 | 70% |
| 1~2년 | 60% |
| 2년 이상 | 기본세율 (6~45%) |
| 1세대 1주택 2년 이상 | 비과세 (9억 이하) |

> 단기 매도하면 양도세가 엄청남. 수익 계산에 반드시 포함.

---

## 손익분기점 계산

```
낙찰가 + 모든 비용 = 손익분기점 매도가
```

이 가격 이상으로 팔아야 본전.

**예시:**
```
낙찰가         2억 5천
취득세          250만
수리비          500만
법무사          80만
이자 (6개월)    375만
중개수수료      예상 270만
= 손익분기점   약 2억 6,475만
```

시세 2억 8천이면 → 약 1,500만 수익.

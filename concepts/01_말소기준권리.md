# 말소기준권리

## 핵심 개념

낙찰 후 등기부등본의 권리들이 **사라지는 기준점**이 되는 권리.

말소기준권리보다 **나중에** 설정된 권리 → 낙찰 시 자동 말소 (소멸)
말소기준권리보다 **먼저** 설정된 권리 → 낙찰자가 인수 (떠안음)

---

## 말소기준권리가 되는 것들

우선순위 순서대로 아래 중 **가장 먼저 등기된 것**이 말소기준권리가 됨.

| 권리 | 설명 |
|------|------|
| (근)저당권 | 가장 흔함. 은행 대출 시 설정 |
| 담보가등기 | 가등기 중 담보 목적인 것 |
| 압류 | 세금 체납 등으로 국가/지자체가 설정 |
| 가압류 | 채권자가 임시로 묶어놓은 것 |
| 경매개시결정등기 | 법원이 경매 시작하며 설정 |

---

## 실제 예시

등기부등본에 이런 순서로 권리가 있다고 가정:

```
2020.01.15  전세권 (세입자 A, 1억)
2021.03.10  근저당권 (○○은행, 2억)   ← 말소기준권리
2022.05.20  가압류 (채권자 B, 3천만)
2023.08.01  전세권 (세입자 C, 8천만)
```

낙찰 시:
- 근저당권(2021) → 말소기준권리, 소멸
- 가압류(2022) → 근저당권 이후 → 소멸
- 전세권 C(2023) → 근저당권 이후 → 소멸
- **전세권 A(2020) → 근저당권 이전 → 낙찰자가 인수!** ← 위험

전세권 A가 1억이면 낙찰자가 1억을 떠안는 것.
이걸 모르고 낙찰받으면 큰 손해.

---

## 초보가 피해야 할 패턴

1. **말소기준권리보다 앞선 전세권/지상권** → 인수 부담
2. **유치권** → 등기부에 안 나옴, 현장 확인 필수
3. **법정지상권** → 토지/건물 소유자 다를 때 발생

---

## 안전한 물건의 조건

- 말소기준권리 앞에 아무것도 없는 것
- 등기부가 단순한 것 (근저당 1~2개, 나머지 후순위)
- 임차인이 있어도 말소기준권리 이후 전입한 것

---

## 체크리스트

- [ ] 등기부등본에서 가장 오래된 권리 확인
- [ ] 말소기준권리 날짜 확정
- [ ] 그 앞에 인수할 권리 있는지 확인
- [ ] 임차인 전입일이 말소기준권리 이전인지 확인

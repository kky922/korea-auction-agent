# 임차인 분석

## 핵심 개념

임차인이 있는 물건은 낙찰 후 **명도(내보내기)**가 필요할 수 있음.
대항력과 우선변제권을 파악해야 명도 난이도와 인수 부담을 알 수 있음.

---

## 두 가지 핵심 권리

### 대항력
- 제3자(낙찰자 포함)에게 "나 여기 살 권리 있어"라고 주장할 수 있는 힘
- 조건: **전입신고 + 점유(실거주)** → 다음날 0시부터 효력 발생
- 대항력 있으면 → 낙찰자가 보증금 돌려줄 때까지 나갈 의무 없음

### 우선변제권
- 배당에서 다른 채권자보다 먼저 받을 수 있는 권리
- 조건: **전입신고 + 점유 + 확정일자**
- 우선변제권 있으면 → 배당 순서에서 확정일자 기준으로 앞에 섬

---

## 임차인 유형별 정리

| 유형 | 대항력 | 우선변제권 | 낙찰자 영향 |
|------|--------|------------|------------|
| 전입 O, 확정일자 O (말소기준권리 이전) | O | O | 보증금 인수 or 배당으로 해결 |
| 전입 O, 확정일자 X (말소기준권리 이전) | O | X | 보증금 인수, 배당 없음 → 위험 |
| 전입 O (말소기준권리 이후) | X | X or 후순위 | 배당 없으면 명도 대상 |
| 전입 X (무단 점유 등) | X | X | 명도 쉬움 |

---

## 실전 판단 흐름

```
임차인 전입일 확인
      ↓
말소기준권리보다 이전?
  ├ YES → 대항력 있음 → 보증금 인수 위험 확인
  │         └ 확정일자도 이전? → 배당 가능 → 배당으로 전액 받으면 OK
  │                              배당 부족 → 차액 낙찰자 인수 or 명도 분쟁
  └ NO  → 대항력 없음 → 배당 여부만 확인
                        배당 있으면 알아서 나감
                        배당 없으면 명도 필요 (비교적 쉬움)
```

---

## 실제 예시

```
2021.03.10  근저당권 ○○은행 (말소기준권리)
2020.11.01  임차인 A 전입, 확정일자 2020.11.01  ← 말소기준권리 이전
2022.06.15  임차인 B 전입, 확정일자 2022.06.15  ← 말소기준권리 이후
```

- **임차인 A**: 대항력 O, 우선변제권 O → 배당에서 근저당보다 앞에 섬
  - 배당으로 전액 받으면 나감, 못 받으면 낙찰자가 차액 줘야 함
- **임차인 B**: 대항력 X → 배당 후순위, 배당 없으면 명도 대상

---

## 명도 난이도

| 상황 | 난이도 | 설명 |
|------|--------|------|
| 임차인 없음 | 최하 | 바로 사용 가능 |
| 후순위 임차인, 배당 전액 | 낮음 | 배당받고 알아서 나감 |
| 후순위 임차인, 배당 없음 | 보통 | 명도소송 없이 협의 가능한 경우 많음 |
| 선순위 임차인, 배당 전액 | 보통 | 배당받고 나가야 하지만 버티는 경우 있음 |
| 선순위 임차인, 배당 부족 | 높음 | 차액 요구하며 버팀, 명도소송 필요 |
| 선순위 임차인, 확정일자 없음 | 최고 | 배당도 없고 대항력만 있음 → 보증금 전액 인수 |

---

## 초보가 피해야 할 패턴

1. **말소기준권리 이전 전입 + 확정일자 없는 임차인** → 보증금 전액 낙찰자 부담
2. **배당으로 보증금 일부만 받는 임차인** → 차액 요구하며 명도 거부
3. **점유자 불명확한 물건** → 현장 답사 필수
